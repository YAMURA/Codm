⚙ CORS Proxy Configuration
⚠
Browsers block cross-origin requests to garena.com.
You need a CORS proxy that forwards cookies and headers. Options:

Option 1 — Local cors-anywhere (recommended):
npx cors-anywhere then set proxy to http://localhost:8080/

Option 2 — Public proxy:
Use https://corsproxy.io/?url= (may not support cookies)

Option 3 — Browser extension:
Install "CORS Unblock" extension and set proxy to empty (direct).
🔐 DataDome Cookie

DataDome is Garena's bot protection. The checker fetches this automatically, but you can also paste a known-good cookie.

ℹ️ About

NIXZ CODM Checker — Web Edition

Telegram: @nixzlls | Owner: @nixzlls


Implements the full Garena authentication flow:

1. DataDome cookie fetch from dd.garena.com

2. Prelogin (get v1, v2 challenge values)

3. Login (MD5 → SHA-256 → AES-ECB password hash)

4. Account info retrieval from account.garena.com

5. CODM OAuth flow (grant → exchange → callback → user info)


All crypto is performed client-side. No data is sent to any server except Garena's APIs (through your CORS proxy).
